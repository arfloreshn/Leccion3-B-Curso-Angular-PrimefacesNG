# Leccion3B

Simple Cliente CRUD hecho con Angular 8.3.6.

# Requisitos
   
  PrimeNG 8 
  Angular 8.3.6
  Node version 10

# Servidores de aplicacion en los que puede correr el cliente

- Payara 5
- GlassFish 5.1
- WidFly 17
- Nginx
- Apache Tomcat 9
